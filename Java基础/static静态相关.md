## static 
- 类变量/静态变量
  - 静态变量会被整个对象所共享
  - 推荐用 类名.类变量名 的格式来访问 也可以用对象名.类变量名 (考虑访问权限)
    - 普通方法则不能通过类名.类变量名 调用
  - 类加载时 类变量出现 类消亡时 类变量死亡
- 类方法/静态方法
  - 不创建对象实例也想调用的方法  可以把方法做成static的 当工具用 提高开发效率
  - 静态方法才可以访问静态变量，也只能访问静态变量 
    - 非静态方法可以访问静态或非静态变量 
  - 类方法中间不能用this super参数的  因为类方法跟具体的对象实例没关系 是个通用的东西
