# list 与 set 区别
|                              list                               |      set       |
|:---------------------------------------------------------------:|:--------------:|
| 可以使用iterator取元素<br> **由于是按进入顺序保存对象<br>能get(int index)下标随机访问元素** |   只能用迭代器遍历元素   |
|                          允许多个Null元素对象                           | 至多允许一个Null元素对象 |
|                             有序 可重复                              |    无序 不可重复     |
